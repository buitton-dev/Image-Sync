# Image Sync
**自分のPCにある画像を、スマホや別端末で安全に見るためのアプリ**

Image Sync は、
**あなたのPCにある画像フォルダを、ブラウザから安全に閲覧するための Windows アプリ** です。

クラウドにアップロードせず、
ポート開放もせず、
必要なときだけアクセスできます。

---

## できること

* 📱 **スマホや別PCで画像を閲覧**
* 🌍 **外出先からも安全にアクセス**（Cloudflare Tunnel使用）
* 🔐 **アクセスコードによる保護**
* 🧹 **不要になったらすべて削除できる**
* ⚡ **インストール後すぐ使える（追加ソフト不要）**

---

## 基本的な使い方（5分で完了）
### 1️⃣ インストール

1. 右側の **[Releases]** (または最新バージョンのリンク) をクリック
2. `Setup.exe` または `.msi` ファイルをダウンロードして実行
3. 特別な設定は不要です

---

### 2️⃣ アプリを起動（管理画面）
起動すると **管理用の画面** が開きます。
ここでは次の操作ができます：

* 画像フォルダを選ぶ
* Web UI を開く
* アクセスコードを管理
* 外部アクセス（トンネル）を開始/停止

※ この画面では **画像は表示されません**（管理専用です）

---

### 3️⃣ Web UI を開く（画像を見る）

管理画面の **「Open Web UI」** を押すと、
ブラウザで画像閲覧画面が開きます。

* PC・スマホ共通のUI
* ログインが必要（アクセスコード）

---

### 4️⃣ スマホから見る（おすすめ）

外出先や別端末から見たい場合：
1. 管理画面で **「Start Tunnel」**
2. 表示された **QRコード** をスマホで読み取り
3. ブラウザでそのまま閲覧

✅ ポート開放不要
✅ HTTPSで暗号化
✅ 停止すれば即アクセス不可

---

## セキュリティについて（重要）

### 🔐 アクセスコード

* 初回起動時に自動生成
* あなたしか知りません
* 変更・再生成が可能

### 🌍 外部アクセス

* Cloudflare の安全なトンネル技術を使用
* あなたのPCのIPアドレスは直接インターネットに公開されません

### 🧯 安全装置

* ログイン失敗が続くと一時的にロック
* HTTPSが使えない場合は外部アクセスを自動停止
* 危険な操作（フォルダ変更など）は管理画面からのみ可能

---

## 「できないこと」も正直に書きます
* ❌ 複数ユーザーでの同時利用（アカウント管理機能はありません）
* ❌ 不特定多数への公開ギャラリー
* ❌ LAN内共有サーバーとしての常時稼働
* ❌ 画像のクラウド保存・同期

このアプリは
**「自分のための安全な閲覧」** に特化しています。

---

## データについて

### 保存されるもの

* 設定（選択したフォルダパスなど）
* キャッシュ（高速表示用のサムネイル等）
* セキュリティ情報（ハッシュ化されたパスワード等）

### 保存されないもの

* あなたの画像ファイル本体
  👉 **元のフォルダや画像ファイルは一切変更・削除されません**

---

## 完全に削除したいとき（重要）

アンインストール前に以下の手順を行うと、ゴミを残さず消せます：

1. 管理画面 → **Settings**
2. **Purge（全データ削除）** を実行
3. その後 Windowsの設定からアンインストール

これで **設定・キャッシュ・アクセスコードは全消去**されます。

## よくある質問

### Q. 常に起動しておく必要はありますか？
→ いいえ。画像を見たいときだけ起動してください。

### Q. URLを知られたら危険ですか？
→ URLだけでは閲覧できません。アプリで設定された「アクセスコード」が必要です。

### Q. 商用利用できますか？
